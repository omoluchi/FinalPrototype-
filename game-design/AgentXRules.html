RULES

Agent X in the Multisensory Universe is an interactive storytelling game that has players incorporate real life objects into the creation of a story.

Each player will find 3 household items before the game:


    OBJECT ALPHA- any object that you can act out and associate with a place and amount of time used.

    OBJECT BETA- an object that generates a sound or makes a noise when otherwise manipulated (tapped, crinkled, dropped, etc.), or both! Pick an object that makes a loud enough sound to be picked up by a mic.

    OBJECT GAMMA- any object that is attached to a personal memory—something where you can clearly tell a story involving the object.


At each stage of the game, players will have to guess what your object is based on various hints based on their primary characteristic. These guesses will contribute to the final generation of Agent X’s adventure!

Remember - This game is about creating a unique universe, so be as creative as possible with your objects and guesses!
